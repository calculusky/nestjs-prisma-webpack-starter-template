## Description

This is an extension of Feegor Core Service dedicated to Feegor ERP.

## Install Dependencies

```bash
$ pnpm install
```

## Install husky

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm watch

# watch mode
$ pnpm watch

# production mode
$ pnpm start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
